## 부동산·세법 글쓰기 스타일 가이드

### 특화 가이드라인
- **실생활 연결**: 추상적 법령을 구체적 사례로 설명
  예: "서울 마포구에 아파트 1채(시가 12억원)를 보유한 A씨의 경우..."
- **세금 계산 예시**: 세율, 공제항목, 계산 과정을 단계별로 보여줌
- **비교표 활용**: 개정 전/후 비교, 지역별 비교를 마크다운 표로 제시
- **주의사항 강조**: 중요한 유의점은 별도 박스로 구분

### 용어 풀이 예시 (괄호 설명 반드시 포함)
- 공시가격: "정부가 공식적으로 발표하는 부동산 가격으로, 세금 부과의 기준이 됨"
- 종합부동산세: "일정 금액 이상의 부동산을 보유한 사람에게 추가로 부과하는 세금"
- 양도소득세: "부동산을 팔아서 생긴 이익(양도차익)에 대해 내는 세금"
- 취득세: "부동산을 살 때(취득할 때) 내는 세금"
- 공정시장가액비율: "공시가격에서 실제 세금을 계산할 때 적용하는 비율"
- 1세대 1주택 비과세: "가구당 주택 1채만 보유 시 양도세를 면제해주는 제도"

### 세금 계산 예시 형식
```
[예시] 서울 아파트 1채 보유자 A씨의 종합부동산세

- 공시가격: 15억원
- 기본 공제: 12억원 (1세대 1주택)
- 과세표준: 15억 - 12억 = 3억원
- 세율: 0.5% (3억 이하 구간)
- 종부세액: 3억 × 0.5% = 150만원
```

### 구성 템플릿 예시
1. [도입] 정책/법 변화의 핵심과 영향 받는 대상
2. [변경 내용] 개정 전 vs 개정 후 비교 (표)
3. [구체적 사례] 가상 사례를 통한 세금 계산
4. [대상별 영향] 1주택자 / 다주택자 / 무주택자 등 유형별 분석
5. [시행일과 경과조치] 언제부터 적용되는지, 유예기간 여부
6. [실전 대응] 독자가 취할 수 있는 구체적 행동
