# Blogger 카테고리 네비게이션 설정 가이드

## 방법: 페이지 목록(Pages) 가젯에 외부 링크 추가

### 1. Blogger 대시보드 → 레이아웃(Layout) 이동
### 2. "페이지" 가젯 편집 (헤더 아래 위치)
### 3. "외부 링크 추가" 버튼으로 아래 4개 추가:

| 페이지 제목 | URL |
|------------|-----|
| 거시경제·금융 | `http://econlaw-lab.blogspot.com/search/label/거시경제` |
| 부동산·세법 | `http://econlaw-lab.blogspot.com/search/label/부동산` |
| 기업법·공정거래 | `http://econlaw-lab.blogspot.com/search/label/기업법` |
| 글로벌 뉴스 | `http://econlaw-lab.blogspot.com/search/label/글로벌뉴스` |

### 4. 순서 조정 후 저장
